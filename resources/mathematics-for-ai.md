# Mathematics for AI and Machine Learning
Essential mathematical foundations including linear algebra, calculus, probability, and statistics for understanding AI and ML algorithms.

## Topics Covered
- Linear algebra (vectors, matrices, eigenvalues)
- Calculus (derivatives, gradients, optimization)
- Probability theory and statistics
- Optimization theory
- Information theory

## Resources

- [Mathematics for Machine Learning (Free PDF Book)](https://mml-book.github.io) - Comprehensive free textbook by Deisenroth, Faisal, and Ong covering linear algebra, calculus, and probability foundations for ML.
- [Mathematics for Machine Learning - GitHub Collection](https://github.com/dair-ai/Mathematics-for-ML) - Curated collection of free resources including books, courses, and tutorials. Features MIT courses, Khan Academy content, and Deep Learning book math chapters.
- [3Blue1Brown - Mathematics Visualized](https://www.youtube.com/c/3blue1brown) - Beautiful visual explanations of linear algebra, calculus, and neural network mathematics through animated videos.
- [Khan Academy - Linear Algebra](https://www.khanacademy.org/math/linear-algebra) - Free comprehensive course covering vectors, matrices, transformations, and other essential linear algebra concepts for ML.
- [Khan Academy - Calculus](https://www.khanacademy.org/math/calculus-home) - Free courses on precalculus, differential calculus, integral calculus, and multivariate calculus.
- [Deep Learning Book - Math Basics](https://www.deeplearningbook.org/contents/part_basics.html) - Free chapters covering applied math and machine learning basics from Goodfellow, Bengio, and Courville.
